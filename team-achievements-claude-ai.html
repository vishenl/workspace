Stylesheet Architecture Cleanup

We used Claude AI to extract, clean, and optimize the entire Mindvalley Academy CSS, reducing bloat and creating a maintainable design system.

üé®

The Challenge

The Mindvalley Academy page used a complex Nuxt.js application with Brotli-compressed CSS split across 10 external files totaling 740KB.

  • Duplicate styles across multiple components
  • Unused CSS bloating the bundle
  • Inconsistent naming conventions
  • No clear design token system
‚ú®

The Solution

Claude AI helped us build extraction tools and clean the CSS systematically:

  • Decompressed Brotli-encoded CSS files with Node.js
  • Extracted design tokens into JSON format
  • Created modular, reusable CSS components
  • Built clean CSS build in mindvalley-clean.css
