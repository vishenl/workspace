# New Mindvalley Self Reset Page

You are creating a new page in the Mindvalley Self Reset design system.

## STRICT REQUIREMENTS - DO NOT SKIP:

1. **Read these files FIRST**:
   - `mv-selfreset-base.css`
   - An existing page like `spiritual-mastery-quiz.html` or `index.html` (if it exists)

2. **CSS Foundation**:
   - Link to `mv-selfreset-base.css` in `<head>`
   - NEVER add custom font-face declarations
   - Use only Grotesk font families from base CSS

3. **Layout**:
   - Wrap all main content in `.mv-container`
   - Use inline padding on sections: `style="padding: 6rem 0 5rem 0;"`

4. **Typography - Use ONLY these classes**:
   - `mv-type--headline1` for hero headings
   - `mv-type--headline2` for major sections
   - `mv-type--headline3/4` for subheadings
   - `mv-type--headline5-9` for smaller labels
   - Plain `<p>` for body text (already styled in base CSS)

5. **Colors**:
   - Primary brand purple: `#7a12d4`
   - Use for links, primary CTAs, accents
   - NO other brand colors

6. **Buttons**:
   - Primary CTA: `background: #7a12d4; color: white; border-radius: 999px; font-family: Grotesk-Bold`
   - Secondary CTA: `background: transparent; border: 2px solid #7a12d4; color: #7a12d4; border-radius: 999px; font-family: Grotesk-Medium`

7. **Common Patterns**:
   - Hero: eyebrow text + mv-type--headline1 + supporting paragraph + CTA buttons
   - Feature grids: `display: grid; grid-template-columns: repeat(auto-fit, minmax(260px, 1fr)); gap: 2rem;`

## FORBIDDEN:

- ❌ Custom font-face declarations
- ❌ New design systems or utility frameworks
- ❌ Different font families
- ❌ New brand colors
- ❌ Removing mv-selfreset-base.css link

## PROCESS:

1. Ask user for page name and purpose
2. Read mv-selfreset-base.css to understand the system
3. Create page following ALL rules above
4. Show user the file and confirm it matches design system
5. Deploy to GitHub when approved

Now ask the user what page they want to create.
