# Workspace Project - Claude Code Instructions

## Design System

**ALL web pages in this workspace MUST use the Mindvalley Self Reset Design System.**

### Before Creating or Editing ANY HTML Page:

1. **Read** `mv-selfreset-base.css`
2. **Read** `docs/mindvalley-design-system.md`
3. **Follow** all typography, layout, and color rules defined there

### Quick Rules:

- ✅ Link to `mv-selfreset-base.css` in every HTML file
- ✅ Use `.mv-container` for layout
- ✅ Use `mv-type--headline1` through `mv-type--headline9` for headings
- ✅ Use Grotesk font families (already in base CSS)
- ✅ Use `#7a12d4` for primary brand color
- ❌ NEVER add custom `@font-face` declarations
- ❌ NEVER override Grotesk fonts
- ❌ NEVER introduce new design systems

### Slash Commands Available:

- `/newpage` - Create a new page following the design system (auto-enforces all rules)

### Reference Files:

- `mv-selfreset-base.css` - Core design system CSS
- `docs/mindvalley-design-system.md` - Complete design system documentation
- `spiritual-mastery-quiz.html` - Example page using the design system correctly

## Project Context

This workspace contains:
- Marketing pages and landing pages for Mindvalley
- Educational curriculum content
- Campaign materials
- Reports and analyses

All public-facing HTML pages should follow the Mindvalley Self Reset design system for brand consistency.
