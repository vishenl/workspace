How do I use these components in my own pages?

+

Simply copy the CSS from the Airtable component library and paste it into your page's style section. Then copy the HTML structure and customize only the content (text, images). Keep all class names and HTML structure exactly the same for best results.

Can I customize the colors and styles?

+

Yes! Use CSS variables like var(--mv-purple) instead of hardcoded colors. You can change the color values in the Design Tokens section at the top of your CSS, and all components will update automatically. This maintains consistency across your entire page.

What if I need a component that doesn't exist?

+

First, check if you can combine existing components to achieve your goal. If you truly need something new, build it following the same patterns: use CSS variables, maintain responsive design, include hover states, and document it in Airtable for future reuse.
