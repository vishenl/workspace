Badge Variants

Class Background Text Color Use Case
.mv-badge--mastery #dafd71 (lime) #181d26 (dark) Primary badges
.mv-badge--dark #061e27 (dark) #dafd71 (lime) Light backgrounds
.mv-badge--white #fff #181d26 Dark backgrounds
.mv-badge--outline Transparent #dafd71 Subtle labels
.mv-badge--md Modifier - Medium size
.mv-badge--lg Modifier - Large size
.mv-badge--floating Modifier - Absolute positioned

Features

  • ✅ Pill-shaped design with 100px border-radius
  • ✅ Uppercase text with letter-spacing
  • ✅ Three size variants (small, medium, large)
  • ✅ Four color variants (mastery, dark, white, outline)
  • ✅ Floating variant for image overlays
  • ✅ Box shadow on floating badges
  • ✅ Inline-flex display for icons + text

CSS Classes Reference

  • .mv-badge - Base badge styles (required)
  • .mv-badge--mastery - Lime green badge (#dafd71)
  • .mv-badge--dark - Dark badge with lime text
  • .mv-badge--white - White badge for dark backgrounds
  • .mv-badge--outline - Transparent with lime border
  • .mv-badge--md - Medium size modifier
  • .mv-badge--lg - Large size modifier
  • .mv-badge--floating - Absolute positioning for overlays
