CSS Classes

  • .mv-footer__container - Centered content wrapper (max-width 1200px)
  • .mv-footer__logo - Footer logo (32px height, centered)
  • .mv-footer__links - Flex navigation links list
  • .mv-footer__link - Individual link styling with hover
  • .mv-footer__divider - Horizontal divider line
  • .mv-footer__copyright - Copyright text (small, muted)
