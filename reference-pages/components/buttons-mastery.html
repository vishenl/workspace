Button Variants

Class Background Text Color Use Case

Features

  • ✅ Hover lift effect: translateY(-1px)
  • ✅ Box shadow on hover for depth
  • ✅ Active state resets transform
  • ✅ Smooth 0.2s transitions
  • ✅ 600 font-weight for all buttons
  • ✅ 0.5rem border-radius
  • ✅ Consistent padding system

Design Tokens

/* Base Button */
padding: 0.75rem 1.5rem
font-size: 1rem
font-weight: 600
border-radius: 0.5rem
transition: all 0.2s ease

/* Small Size */
padding: 0.625rem 1.25rem
font-size: 0.875rem

/* Hover States */
transform: translateY(-1px)
box-shadow: 0 4px 6px rgba(0, 0, 0, 0.15)

/* Colors */
--mastery-400: #dafd71 (lime green)
--mastery-850: #181d26 (dark text)
--mastery-950: #061e27 (dark background)

CSS Classes Reference

  • .base-button - Base button styles (required)
  • .mv-btn--mastery - Primary mastery button
  • .mv-btn--mastery--small - Small size modifier
  • .mv-btn--mastery-white - White outline for dark backgrounds
  • .mv-btn--mastery-secondary - Secondary outline button
  • .mv-btn--white - White filled button
  • .mv-btn--dark - Dark button for light backgrounds
